## 📖 About

The **Library Management Portal** is a full-stack data application built as part of a group project at **WBS Coding School**. It provides a simple, intuitive interface for managing a personal library — tracking books, registered friends, and loan activity — all backed by a MySQL database.

---

## ✨ Features

### 📊 Dashboard
- Overview of total books, available books, registered friends, and active loans
- Live table of books currently out on loan

### 📚 Books Management
- Browse the full book inventory
- Add new books with title, author, ISBN, genre, and comments
- Update book details and availability status
- Remove books from inventory (with active loan protection)

### 👥 Friends Management
- View all registered friends
- Add new friends to the directory
- Update friend contact details and loan limits
- Remove friends from the directory

### 🔄 Loan Register
- View all active and past book loans
- Issue books to registered friends
- Update loan status (Borrowed, Returned, Overdue)
- Log return dates and remarks
- Remove loan records
